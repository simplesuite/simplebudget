simpleBudget is a Progressive Web App (PWA), which means you can install it directly from your browser — no App Store needed.

## Steps

1. **Open Safari** and go to [app.simplebudget.com](https://app.simplebudget.com)
2. Tap the **Share button** (the square with an arrow pointing up) at the bottom of the screen
3. Scroll down and tap **"Add to Home Screen"**
4. Give it a name (or keep "simpleBudget") and tap **Add**

That's it. You'll see the simpleBudget icon on your home screen, and it will open in its own window — no browser toolbar, no tabs.

## Tips

- You need to use **Safari** — this doesn't work from Chrome or Firefox on iOS
- Once installed, the app works offline for viewing your budget. Transactions sync when you're back online
- To uninstall, long-press the icon and tap "Remove App" just like any other app

## Troubleshooting

**"Add to Home Screen" isn't showing up?**
Make sure you're using Safari (not an in-app browser) and that you're tapping the system Share button, not a share button within the page.

**The app looks different after updating?**
If you see an update prompt inside the app, tap "Update" to get the latest version. The PWA caches itself for offline use, so updates need a manual refresh.
